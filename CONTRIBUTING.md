# 🤝 Contributing to SE Audio Converter

Thanks for helping improve `universal-audio-converter`.

---

## 🐞 Bugs and feature requests

Please open a GitHub issue for:

- conversion or export bugs
- editor/UI problems
- `Audio.sbc` or `SoundBlock.sbc` generation issues
- setup or packaging problems
- documentation fixes and improvement ideas

If possible, include the input format and the exact result you expected.

---

## 🔀 Pull requests

PRs are welcome.

Please:

1. keep PRs narrowly scoped
2. document any workflow or dependency changes
3. avoid committing generated files or local-only notes
4. call out anything that could affect existing mod setups

---

## 👀 Review flow

This repo uses code ownership and auto-assignment so **`@Godimas101`** is requested on pull requests and tagged as the default maintainer contact.

---

*If it saves modders from wrestling audio by hand, it's probably worth the PR.*
